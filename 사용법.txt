【 처음 설치 시 】
-----------------------------------------------------------------
1. Spring-Toever-Ops-Setup-x.x.x.exe 실행 → 설치 완료
2. 바탕화면 "Spring Toever Ops" 아이콘 실행
3. 첫 실행 시 초기 설정 마법사 화면이 뜹니다
   → 투에버 ID / 비밀번호 입력
   → 저장소 경로 지정 (기본값 사용 권장)
   → 저장 버튼 클릭


【 매일 사용 순서 】
-----------------------------------------------------------------

[STEP 1] 주문 수집
  - 대시보드 → "주문 수집 날짜" 선택 (보통 오늘 날짜)
  - "주문 수집 실행" 버튼 클릭
  - 브라우저가 자동으로 열리고 투에버에 로그인하여 주문을 수집합니다
  - 완료 메시지 확인 (예: "수집 완료: 총 5건, 신규 3건")
  - ※ 수집은 하루에 여러 번 해도 됩니다 (중복 제외됩니다)

[STEP 2] 이지어드민 업로드 파일 생성
  - 대시보드 → "업로드 파일 생성" 버튼 클릭
  - 엑셀 파일이 자동 생성되고 탐색기에서 해당 폴더가 열립니다
  - 생성된 엑셀 파일을 이지어드민에 직접 업로드하세요

[STEP 3] 송장 임포트 (이지어드민 송장 처리 후)
  - 이지어드민에서 송장 처리 완료된 엑셀 파일 준비
  - 송장 관리 화면 → "이지어드민 송장 파일 선택" 버튼 클릭
  - 파일 선택 → 임포트 실행
  - 완료 메시지 확인 (예: "5건 처리")

[STEP 4] 투에버 송장 업로드 (하루 1회)
  - 송장 관리 화면 → 대기 중인 송장 건수 확인
  - "업로드 실행" 버튼 클릭
  - 브라우저가 자동으로 열리고 투에버에 송장을 업로드합니다
  - 완료 메시지 확인 (예: "5건 송장 업로드")
  - ※ 투에버 송장 업로드는 하루에 1번만 합니다
  - ※ 오늘 이미 업로드했으면 경고 메시지가 표시됩니다


【 자주 있는 상황 】
-----------------------------------------------------------------

▶ "투에버 ID/비밀번호가 설정되지 않았습니다" 오류
  → 설정 화면에서 투에버 ID/비밀번호 입력 후 저장하세요

▶ "이미 수집된 데이터입니다" 안내
  → 해당 날짜 주문이 이미 수집됨. 정상입니다 (새로운 주문 없음)

▶ "로그인 실패" 오류
  → 투에버 ID/비밀번호가 맞는지 확인하세요
  → 투에버 사이트(https://support.toever.co.kr)에서 직접 로그인 테스트해보세요

▶ 브라우저가 켜졌다가 꺼지거나 깜빡이면
  → 설정 화면 → "브라우저 설정" 섹션에서 Chromium 상태 확인
  → Chromium 설치 버튼이 보이면 클릭하여 설치

▶ "저장소에 접근할 수 없습니다" 오류
  → 설정 화면에서 저장소 경로 확인 (외장 드라이브 연결 여부 확인)

▶ 앱이 느리거나 먹통이면
  → 작업 관리자에서 "Spring Toever Ops" 프로세스 모두 종료 후 재실행


【 설정 화면 안내 】
-----------------------------------------------------------------
  투에버 계정    : 투에버 협력업체 ID / 비밀번호
  저장소 경로    : 주문·송장 파일이 저장되는 폴더 경로
  백업 경로      : 자동 백업 저장 위치 (외장 SSD 권장)
  스케줄러       : 자동 수집 시간 설정 (사용하지 않으면 비활성화)
  Chromium       : 브라우저 자동화 엔진 (최초 설치 필요할 수 있음)


【 다른 PC로 이전 시 】
-----------------------------------------------------------------
1. 기존 PC에서 수동 백업 실행 (설정 → 백업/복구 → 백업 실행)
2. 백업 폴더를 USB/외장드라이브에 복사
3. 새 PC에 설치 파일(exe) 설치
4. 설정 → "백업 드라이브 선택 후 복구" 클릭하여 복원
5. 앱 재시작
